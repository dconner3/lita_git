Getting to Know Git: A Guide to the ALA Class

This is a resource guide for the upcoming ALA session, "Getting to Know git". In this class we will talk about what git is, why you should use it, and give you a brief lesson on how to work with git.

We will save the last half hour to have people investigate git (and GitHub), with different steps depending on whether you want to use command-line (command-line is strongly encouraged but not necessary).

Feel free to follow any of these steps before class!

  • Set up with GitHub
  • Using Git Without the Command-line
  • Using Git With the Command-line
  • Resources for learning more about Git
