Getting to Know Git: A Guide for the ALA Class

This is a resource guide for the upcoming ALA session, "Getting to Know git". In this class we will talk about what git is, why you should use it, and give you a brief lesson on how to work with git.

We will save the last half hour to have people investigate git (and GitHub), with different steps depending on whether you want to use command-line or not (command-line is strongly encouraged but not necessary).

Feel free to follow any of these steps before class!

  • Set yourself up with GitHub - see instructions below (btw, you don't need to use GitHub to use git, but it does give you a lot of benefits).
  • Use Git Without the command-line
  • Use Git With the command-line
  • Resources for learning more about Git

GitHub

You'll probably want to use GitHub as a space to view and store your own projects and view other people's.

  1. Get a user name and password: https://github.com/join
  2. Look around at the repositories already on GitHub. You might recognize a few:
    • JQuery
    • NYPublic Library
    • Fedora
    • The code for these slides
